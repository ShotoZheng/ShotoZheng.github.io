发表于 2021-06-06 更新于 2021-07-03 分类于 项目 ， 谷粒商城 阅读次数： Valine：

1. SpringCloud Alibaba 集成

  • 集成 Nacos 作为注册中心
  • 采用 OpenFeign 方式进行 RPC 操作
  • 集成 Nacos 作为配置中心
  • 采用 SporingCloud Gateway 作为网关，实现路径重写和负载均衡等

2. 前端了解

  • ES6 语言基础了解
  • VUE 基础了解
  • ElementUI 基础了解

3. 三级分类开发

3.1 网关路径重写和跨域问题

  • 前端 renren-fast-vue 项目统一后台网关访问路径，比如http://localhost:88/api/ + 模块标识，如 product,order 等，后台 gulimall-gateway 网关项目则对不同的访问路径进行重写，并分发到对应的功能模块中，比如前端访问路径为 http://localhost:88/api/product/* ，则会被重写为 http://localhost:88/product/* 并负载均衡访问 gulimall-product 项目模块

  • 由于前后台端口不一致而导致跨域问题，对应解决方法查看 cors 跨域问题处理 该文章

3.2 三级分类树查询功能

数据模型：pms_category

  • 查询渲染三级分类树，需要从后台查询出对应的树形数据结构数据，对应的解决方法查看 Java 实现树形数据查询 该文章
  • 分类删除功能开发（MyBatis-Plus 逻辑删除）
  • 分类新增功能开发
  • 分类修改、批量修改功能开发

5. 平台属性

数据模型：pms_attr_group(属性分组)、pms_attr(规格参数和销售属性)

  • 属性分组
    • 实现属性分组 attr_group 属于一个分类 category
    • 实现属性分组 attr_group 关联未关联的属性 attr
  • 规格参数
    • 实现属性分类和所属该类下的分组
  • 销售属性
    • 实现属性分类和所属该类下的分组
