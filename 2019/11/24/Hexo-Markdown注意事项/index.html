3. 显示阅读全文

Hexo 首页默认是会显示每篇文章的全部内容的，为了更为合理的浏览首页的内容，我们无需显示整篇文章，而是增添一个阅读全文的按钮，并只在首页显示文章指定的部分内容，若要查看该篇文章则只需点击阅读全文即可。
