1. 静态方法

在 Java 8 中，允许在接口中增加静态方法。理论上讲，这是合法的，只是这有违于将接口作为抽象的初衷。

目前为止，通常的做法是将静态方法放在伴随类中，比如标准库中，有成对出现的接口和对应工具类，如Path/Paths，Colletion/Collections。
