在Java中有种动态代理技术，比如JDK，CGLIB，Javassist，ASM，其中最常用的动态代理有两种：一种JDK动态代理，这是JDK自带的功能；另一种CGLIB，这是第三方提供的一个技术。目前，Spring常用JDK和CGLIB，而MyBatis还使用了Javassist，无论哪种代理其技术和理念都是相似的。

下面讲一下常用的JDK动态代理和CGLIB动态代理。
