IDEA中Git的使用

发表于 2020-01-01 分类于 Git 阅读次数： Valine：
本文字数： 758 阅读时长 ≈ 1 分钟

引言

本篇文章讲述在IDEA开发工具中git的详细使用过程,提高开发效率

IDEA配置git

1.电脑必须已经装好了git 的客户端git客户端 .下载地址git
2.打开IDEA设置界面找到git选好git安装的路径点击Test测试如果出这个提示就是成功了

使用详情

把项目上传到本地库和远程库

1.为新建的项目添加本地git库

2.选择项目根目录点击确定

3.可以看到现在文件都是红色的说明还没被跟踪

4.把所有文件跟踪添加到stage区文件都变成了绿色

5.提交到本地库远程库

6.可以在commitmessage里面写本次提交的详情然后Commit and Push


7.第一次提交到远程库需要配置一下远程库的地址输入账号密码点Push提交


Git图解：

从远程Git仓库获取项目源码

1.如果有人要下载项目源码需要先配置好git点击Configure>settings剩下的按照IDEA配置git的步骤

2.下载项目源码,填写地址下载路径,点击clone下载

获取别人提交的代码提交自己更新的代码

1.在每次提交更新的代码前先把别人提交的代码下载下来

2.提交更新的代码

创建分支并在分支上开发

建分支也是一个常用的操作，例如临时修改bug、开发不确定是否加入的功能等，都可以创建一个分支，再等待合适的时机合并到主干
1.创建一个分支

2.可以看到右下角已经切换到了新创建的分支上面LocalBranches是本地分支RemotBranches是远程分支

3.把新的分支提交到远程库

合并分支

1.如果分支上的代码已经没有问题需要合并到主分支先切换到主分支

2.把分支合并到主分支点击merge合并
